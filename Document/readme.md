# FreelanceFinder

A full-stack freelancing platform built with React and Node.js.

## Features
- 🎯 Post and apply to freelance projects
- 📬 Real-time chat using socket.io
- 🔐 Authentication for Clients and Freelancers
- 📊 Admin dashboard

## Tech Stack
- React.js
- Node.js + Express
- MongoDB
- Socket.io

## How to Run
```bash
cd server
node index.mjs

# In new terminal
cd client
npm start
